# Sistem Integrasi Sumber Daya Ikan

A web-based dashboard prototype developed during the MagangHub Kemnaker Internship at the Directorate of Fish Resource Management, Ministry of Marine Affairs and Fisheries.

## Technologies
- HTML
- CSS
- JavaScript
- Leaflet.js
- Chart.js
- Turf.js
- GeoJSON

## Key Features
- Interactive Web Map
- Layer Management
- Search Function
- Dynamic Filtering
- Data Visualization
- Responsive Dashboard Interface

## Status
Prototype Completed (Development Paused after Internship Ended)
